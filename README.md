# SE能力可視化

SE能力値を入力すると、ステータスカード風に表示できるWebアプリ。

## 技術スタック

- React + Vite + TypeScript
- Tailwind CSS
- PWA (vite-plugin-pwa)
- React Router

## 開発環境のセットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## デプロイ

Vercel を使用。`main` ブランチへのpushで自動デプロイ。

## MVP

- トップ画面
- 能力値入力画面
- ステータスカード表示画面
- Web公開URL
